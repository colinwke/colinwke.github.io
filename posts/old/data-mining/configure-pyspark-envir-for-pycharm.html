配置pyspark本地调试环境

Alphacks 收录于 数据分析
 2017-09-09  约 512 字   预计阅读 2 分钟 

1 下载

https://spark.apache.org/downloads.html 注: 不要追求最新的版本, 最新的版本意味着坑可能多. 示例版本: spark-2.1.1-bin-hadoop2.6.tgz

2 解压

注: 解压后的路径不能包含空格 (尤其注意Program File, 另外, spark依赖java环境, 因此java安装路径也不能有空格!)

3 配置

进入spark根目录下python文件夹, 复制pyspark文件夹至python根目录下Lib\site-packages文件夹内.

4 测试

4.1 设置本地环境

注意: 一定先要设置环境, 再导入spark的包(from pyspark import SparkContext)

import os
import sys
import glob


def init_local_env():
    """ init local spark envir """
    # Path for spark source folder
    SPARK_HOME = 'C:\ProgramDev\spark-2.3.3-bin-hadoop2.6'  #
    HADOOP_HOME = SPARK_HOME + '\hadoop'

    os.environ['SPARK_HOME'] = SPARK_HOME
    os.environ['HADOOP_HOME'] = HADOOP_HOME

    # You might need to enter your local IP
    # os.environ['SPARK_LOCAL_IP'] = '192.168.2.138'

    # Path for pyspark and py4j
    py4j_path = SPARK_HOME + '\python\lib\py4j-*-src.zip'
    py4j = glob.glob(py4j_path)
    if not py4j: raise FileNotFoundError("No `py4j-src` in `%s`!" % py4j_path)
    sys.path.append(py4j[0])


init_local_env()

# init spark env before import spark!
from pyspark import SparkContext
